## 🛠️ Customization Tips

### Change Colors

Edit `css/styles.css`:

```css
:root {
  --color-neon-blue: #00d4ff; /* Cyan accent */
  --color-neon-purple: #a855f7; /* Purple accent */
  --color-neon-pink: #ec4899; /* Pink accent */
}
```

### Update Text

Edit sections directly in `index.html`:

- Hero headline
- Service descriptions
- Contact information
- Footer content

### Add New Section

Follow the existing pattern:

1. Create new `<section>` with ID
2. Add `.section-badge`, `.section-title`, `.section-subtitle`
3. Use appropriate card classes
4. Add `scroll-reveal` for animations
