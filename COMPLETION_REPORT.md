## 🔧 CUSTOMIZATION GUIDE

### Change Brand Colors

Edit `css/styles.css`:

```css
:root {
  --color-neon-blue: #00d4ff; /* Your color */
  --color-neon-purple: #a855f7; /* Your color */
  --color-neon-pink: #ec4899; /* Your color */
}
```

### Update Content

Edit `index.html` sections:

- Hero headline & subheading
- Feature descriptions
- Service details
- Contact information
- Footer links

### Add New Section

1. Create new `<section>` with ID
2. Use `.section-badge`, `.section-title`, `.section-subtitle`
3. Use appropriate card classes (`.glass-card`, `.service-card`, etc.)
4. Add `scroll-reveal` class for animations

### Update Logo

Replace `assets/images/logo.svg` with new logo maintaining:

- Same SVG structure
- Gradient IDs
- Accessibility labels
- Professional appearance
